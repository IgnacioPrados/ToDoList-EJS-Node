# TodoList-EJS-Node
ToDoList created with EJS Template, Node and Express
<br><br>
 Creator: Ignacio Prados.
 <br><br>
 Key Technologies used: HTML5 - CSS3 - JavaScript - EJS - Node.js - Express.js
 
## Installation
 ```
 npm install
 ```
 ## Run server
 ```
 nodemon start
 ```
 or
 ```
 npm start
 ```
